# Pokedex

A simple and responsive Pokedex built with the [PokeAPI](https://pokeapi.co/).

## Features

- Loads Pokemon from the API
- Search by Pokemon name
- Type colors and icons
- Detail view with information, stats and evolution chain
- Load more button
- Responsive design

## Technologies

- HTML
- CSS
- JavaScript

This project was created for the Developer Akademie Frontend Course 2026.
